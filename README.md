# IoT-Firstep
一个物联网(IoT)开发的入门教程。涉及单片机、上位机、移动应用、服务器后台开发的知识。以及蓝牙4.0、以太网模块的使用实例。
## 状态
努力编写中，还未进行核查。。。
## 说明
本文的开发环境都是在Windows平台进行的。关于iOS相关的开发暂时没有编写，后续可能会添加，章节编号4.x暂时空出来。
