# IoT-Firstep
一个物联网(IoT)开发的入门教程。涉及单片机、上位机、移动应用、服务器后台开发的知识。以及蓝牙4.0、以太网模块的使用实例。
## 更新计划
计划iOS使用swift语言重构，Android部分将基于Android Studio重新编写。
## 状态
努力编写中，还未进行核查。。。
## 说明
本文的开发环境除了iOS的开发，都是在Windows平台进行的。
