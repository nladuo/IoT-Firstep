# IoT-Firstep
一个物联网(IoT)开发的入门教程。涉及单片机、上位机、移动应用、服务器后台开发的知识。以及BLE、WiFi、GSM、MQTT协议的使用实例。
## 状态
努力编写中。。。
## 说明
本文的开发环境除了iOS的开发，都是在Windows平台进行的。
